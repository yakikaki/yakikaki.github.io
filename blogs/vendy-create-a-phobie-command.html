How does it work?

Once you run the command you're gonna be prompted with a menu that looks like this:


each button opens a different menu with 5 questions each! the bot will generate you a custom card based on the answers you've wrote inside the menus. easy right?
lets go over each menu and see what questions it holds.


Configure Base Stats

This menu holds the general card data, such as:

  • What will be the name of the card? - Example: Jeremy

  • How much keys does it cost? - Cards vary between 1-9 keys

  • What race is the card? - either Dimensional / mechanical / monster / undead

  • What will be the card's catchphrase? - Some cards have a catch phrase for example razor mouth: Here Kitty Kglllahhhhhhh!

  • What rarity is the card - Either common / uncommon / rare / ultra rare


Configure Number Stats

This menu holds the statistics data, such as:

  • How much health does the card have? - any number

  • How much damage does the card deal? - any number

  • How much movement range does the card have? - any number

  • What level is the card? - any number (cards usually vary between 1 through 30)

  • what damange range does the card have? - Any mumber


Configure Extra Stats

This menu holds the types of each parameter, it mainly just changes the icons of the stats:

  • What attack type does the card have? -Line of Sight / Lob

  • What target Range will card have - Single target attack / AOE / Double / self / line (aka beam)

  • what type of movement will the card have? - walking / bypass (burrow) / flying
